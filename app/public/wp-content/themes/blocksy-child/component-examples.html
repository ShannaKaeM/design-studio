How Studio Components Work

1. CSS Variables (Design Tokens)

/* s-vars.css */
--s-primary: #0066cc;
--s-space-4: 1rem;
--s-radius-1: 0.25rem;

All styling comes from centralized variables that you can change globally.

2. Component Classes

/* s-selectors.css */
.s-btn {
  padding: var(--s-space-3);
  background: var(--s-primary);
}

Components use variables, so changing a variable updates all components.

3. Utility Classes

/* Auto-generated */
.s-text-primary { color: var(--s-primary); }
.s-p-4 { padding: var(--s-space-4); }
.s-rounded { border-radius: var(--s-radius-1); }

Utilities provide quick styling without writing custom CSS.

4. Use in WordPress

<!-- In Gutenberg blocks -->
<div class="s-card">
  <h3 class="s-card__title">Title</h3>
  <p class="s-card__content">Content</p>
</div>

Apply component classes to any HTML element or WordPress block.
