== Description ==

The Studio is a groundbreaking WordPress theme that implements Daniel's innovative variable-driven design system. It features:

* CSS variables that auto-generate UI controls
* Selector builder to target any element
* AI-friendly YAML content management
* ACF Pro integration with Local JSON
* Seamless GenerateBlocks compatibility

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through the 'Themes' screen in WordPress
3. Install and activate ACF Pro (required)
4. Visit The Studio menu in the admin
